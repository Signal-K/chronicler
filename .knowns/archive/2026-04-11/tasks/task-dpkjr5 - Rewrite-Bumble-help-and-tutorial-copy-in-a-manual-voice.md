# Rewrite Bumble help and tutorial copy in a manual voice

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Provide a revised help/tutorial copy set that reads like a clear in-game manual instead of character dialogue. Keep the tone cosy but practical, remove references that overpromise on hidden systems, and align tutorial/help text with current design decisions such as short hatch moments, simple progression, and retired bees as the eventual production sink.
