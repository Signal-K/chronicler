# Web/Next and Electron Godot host shells

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Create web host surfaces for Godot export so the same game project can run inside React/Next and Electron shells.

Acceptance criteria:
- Godot web export is embedded and responsive in browser host page.
- Electron window can boot and display the same exported Godot build.
- Host shell provides basic bridge events (focus, pause, resume, quit).
